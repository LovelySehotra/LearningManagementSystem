# LMS Frontend

### Setup instruction

1. Clone the project

```
git clone https://github.com/LovelySehotra/LearningManagementSystem.git
```

2. Move into the directory 

```
cd lms-frontend

```
3. install dependencies

```
npm i

```
4. run the server

```
npm run dev

```
